IK Multimedia iKlip 3 Video

■商品紹介

★あなたのクリエイティビティをサポート

IK Multimediaの最新のカメラ三脚用マウント、iKlip 3 Videoをご紹介しましょう。iKlip 3 Videoならクリエイティブなシャッター・チャンスを決して逃しません。IKは7年以上もの間モバイル・デバイス・マウントのパイオニアであり続けてきました。さらに改良されたiKlip 3 Videoは、これまで以上にあなたのクリエイティビティをより素早く、簡単に、信頼できる方法でサポートするソリューションとなるでしょう。

重たい撮影機材を持ち運ぶのはとても疲れることでしょう。iKlip 3 Videoなら、あなたは軽量かつ安心できる方法でiPadやタブレットを写真撮影、ビデオ収録などに活用でき、さらにはその場で編集してアップロードしたりシェアしたりすることができるのです。

●固定方法は自由自在
iKlip 3の新たなデザインでは、3つの固定方法に対応しています。どんなマイク・スタンドにも正面、横向きに固定でき、ブーム・スタンドへの取り付けも可能です。常にパフォーマンスに最適な位置へとデバイスを固定できます。

マイクスタンドへ取り付けるホルダーとセットになったiKlip 3、カメラ用三脚に取り付けるホルダーとセットになったiKlip 3 Video、そしてその両方のホルダーをセットにしたiKlip 3 Deluxeの3つのラインナップを用意。

◎iKlip 3・・・iKlip 3 ブラケットとマイクスタンド用マウントホルダーとのセット
◎iKlip 3 Video・・・iKlip 3 ブラケットと三脚用マウントホルダー及びUNC 1/4-20オス - 5/8- 27メスアダプターとのセット
◎iKlip 3 Deluxe・・・※iKlip 3 ブラケットとマイクスタンド用マウントホルダー、三脚用マウントホルダー及びUNC 1/4-20オス - 5/8-27メスアダプターとのセット

充実した固定方法により、ステージやスタジオでの音楽シーンでの活用だけでなく、プレゼンテーションや、オーディオ、ビデオの収録など、あらゆる領域でクリエイティブにタブレットを活用していただくことが可能です。

●安心して固定していただくために
iKlipシリーズは30万本近くの販売実績があり、これまで数えきれないほどの数のタブレットを支えてきました。私たちは長年の経験を経たイタリアン・デザインから生み出される高品位で高性能な製品に誇りをもっており、タブレットのような高価で大切なデバイスを、安価な素材で危険に晒すことはできないと理解しています。

iKlip 3 Videoはサーモ・プラスティックで作られており、頑丈かつ軽量で持ち運びも簡単です。そして新たに、耐久性の高い大きなラバー・グリップを採用しています。これは何年にもわたって着脱を繰り返しても緩んだりすることがなく、そしてもちろん、あなたの大切なデバイスに傷をつけることもありません。新しいデザインは、ケースの有無に関係なくすべての接続用ポートを使用可能としています。

さらに、将来発売される新たなデバイスを見据えた、フレキシブルさも持ち合わせています。iKlip 3 Videoは、あなたのあらゆる冒険の完璧な相棒となるのです。

●様々な固定方法に対応します
iKlip 3 Videoは標準規格のUNC 1/4-20ネジを採用しているので、ほぼすべてのカメラの三脚、一脚やその他のマウント方法に固定でき、その柔軟性により無限の可能性を提供します。あなたの思い描く様々な活用方法に対応できることでしょう。

さらにiKlip 3 Deluxeなら、直径1.2インチ（30 mm）までのポールにも固定することができます。このポール・マウントは数秒で設置でき、ポールの正面または側面に水平と垂直の両方に取り付けることができます。そのため、どのような理想のアングルでもデバイスをマウントできます。

5つの充実した固定方法により、ステージやスタジオでの音楽シーンでの活用だけでなく、プレゼンテーションや、オーディオ、ビデオの収録など、あらゆる領域でクリエイティブにタブレットを活用していただくことが可能です。

●市場で最もユニバーサルなマウント
iPad miniからSamsung Galaxyまで、ケースの有無にかかわらずiKlip 3 Videoは非常に多くのタブレットに対応します。新たに改善されたクリップは、画面サイズが7インチから12.9インチまでのさまざまなタブレットに対応していますので、あなたがお持ちのデバイスを難なく撮影に活用していただけます。デバイス・ケースを取り外す必要もありません。

市場で最もユニバーサルなタブレット・マウントですので、将来の新しいデバイスへの買い替えにも対応できます。

●シャッター・チャンスを逃しません
シャッター・チャンスは一瞬です。だからこそiKlip 3 Videoは、即座に取り付けできるように設計されています。iKlip 3 Videoのブラケットを使うと、デバイスを素早く固定でき、数回ひねるだけで三脚に取り付け可能です。

iKlip 3 Videoはボール＆ソケット式のジョイント部が採用されていますので、画面が縦向きでも横向きでも常に最適なアングルで撮影することができます。

あなたの撮影チームにiKlip 3 Videoが加われば、一瞬のベスト・ショットも撮影可能となることでしょう。

●撮影の準備は常に万全です
iPadやAndroidタブレットでのビデオ収録が初めてでもご安心ください。IKの無償アプリ、iRig Recorder for iOS、Androidをダウンロードするだけで準備は完了です。このアプリにはプロ・クオリティでの撮影、編集、アップロードを行うためのすべてが備わっています。

iRig Recorderはテキスト＆フォト・マーカー、デバイス上での編集、高度なオーディオ処理、そして簡単にエクスポートやシェアが可能なフル機能のアプリです。さらに、アプリ内でiKlip 3 Videoのシリアル・ナンバーを登録すれば、追加機能もアンロックされます。iKlip 3 Videoは、箱を開けてすぐに必要なものがすべて揃うのです。

iKlip 3 VideoとiRig Recorderの強力な組み合わせによって、高い信頼性と使いやすさを兼ね備えたシステムが実現するのです。さあ、撮影に出掛けましょう！
