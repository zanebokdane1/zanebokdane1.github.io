RATOC RS-EC32-U3RZ 便利なUSB接続!バックアップに最適な3.5インチHDD専用RAIDケース HDD2台用 ブラック


便利なUSB接続!バックアップに最適な3.5インチHDD専用RAIDケース HDD2台用 ブラック

本製品は、3.5インチSATA HDDを2台組み込み可能な、USB接続ハードウェアRAIDケースです。12TB HDD(Western Digital製)にて動作確認済み。
※薄型タイプのHDDには対応していません。
※2.5インチHDD/SSDには対応していません。

■特徴
3.5インチHDD 2台用RAIDケース
RAID1(ミラーリング)など4つのモードを装備
ミラーリングでも160MB/s超の高速転送
HDDの取り付け・セットアップが簡単
スマートデザイン、コンパクト設計
アルミでしっかり放熱、FANコントロールで静音
RAID、省電力、暗号化の管理ソフトを提供

■仕様
動作モード:RAID1(ミラーリング)、RAID0(ストライピング)、JBOD(スパンニング)、SINGLE(2台のHDDを個別に認識、1台のみ装着することも可能)
対応HDD:3.5インチ SATA HDD NCQ(Native Command Queuing)対応、4K Native対応、AFT(Advanced Format Technology)対応(4096バイトセクターに最適化された動作はおこなわれず、従来の512バイトセクターでの動作となります)
対応ファイルシステム:NTFS・FAT32・exFAT・HFS+ ※FAT32をご使用の場合、4GBを超えるファイルは扱えません。また、2TBを超えるボリュームは作成できません
RAIDコントローラ:ソシオネクスト MB86E501A
バスインターフェイス:USB 3.0 Specification Revision 1.0
パソコン用接続コネクタ:USB 3.0 Standard-B
ドライブ用接続コネクタ:SATAシグナルコネクタ7ピン、SATA電源コネクタ15ピン
転送速度(理論値):SuperSpeed 5Gbps、High-Speed 480Mbps
USBクラス:USB Mass Storage Class Bulk-Only Transport
ホットスワップ(RAID1):HDD障害発生時、PCや本製品の電源をOFFすることなく、HDDを交換可
オートリビルド(RAID1):HDD交換後、自動的にリビルドを開始。PC電源をOFFにしても、リビルドは継続。
DIPスイッチ:1:ブザー設定、2:電源連動設定
ブザー:HDD抜挿、HDD異常、RAIDシステム異常の検出時にブザーで通知
電源連動:通常は無効の状態で使用(テレビに接続する場合のみ、有効に設定してください。無効の場合も、パソコンの電源OFF、スリープや休止モード時には、HDDのスピンダウンがおこなわれます。)
セキュリティスロット:搭載(本体背面)
複数台接続:台のPCに本製品を複数台接続して使用可
データ暗号化:ハードウェアAES暗号化(XTSモードサポート)
電源電圧(ACアダプタ):入力:AC 100V、出力:DC +12V/3.0A
消費電力:AC100V 13W(待機時)
HDDへの電源供給能力:HDD1台あたり最大17W(アクセス時) ※DC +12V、+5Vの合計消費電力
外形寸法(W×H×D):81×123×177mm(突起部含まず)
材質:アルミ、プラスチック(ABS樹脂)、鋼板
重量:約770g
添付品:製品本体、ACアダプター(12V/3A)、HDD固定ネジ(2本)、USB 3.0ケーブル(約1m)、ユーザーズマニュアル、保証書(1年)
提供ソフトウェア:RATOC RAID監視マネージャー(RAID監視、暗号化、省電力設定) ※メーカーホームページよりダウンロード提供
対応機種:USB Type-Aポートを装備したWindows PC/Mac
対応OS:Windows 11、Windows10、Windows 8.1、Windows 7、MacOS 10.12.x、MacOS X 10.11.x、MacOS X 10.10.x ※32ビット/64ビット両対応、各OSの最新バージョンにアップデートしてご使用ください。Mac OSとWindowsでデータを共有するためには、exFAT形式でフォーマットしておく必要があります。

■注意事項
USB 3.0(SuperSpeed)で使用するには、USB 3.0対応のインターフェイスが必要です。
RAID0、RAID1で動作させる場合、同一メーカー、同一型番のHDDを使用されることをお勧めします。
HDDは添付していません。
本製品は国内仕様となっており、海外についての保守およびサポートはおこなっておりません。
予告なく外観または仕様の一部を変更することがあります。
