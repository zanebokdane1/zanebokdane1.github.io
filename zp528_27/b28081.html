Nexus Robot Arduino 328マイクロコントローラ

概要

  • DCモータドライバを搭載したArduino Atmega328マイクロコントローラ
  • デジタルI / Oピン：14本（うち6本はPWM出力）
  • フラッシュメモリ32 KB
  • 動作電圧：5V

Nexus Robot Arduino 328マイクロコントローラはArduino 168をベースにしたマイクロコントローラボードで、14個のデジタル入出力ピン（6個はPWM出力として使用可能）、8個のアナログ入力、16 MHz水晶発振器、USB接続、電源ジャック、ICSPヘッダー、およびリセットボタンが含まれています。これには、マイクロコントローラをサポートするために必要なものがすべて含まれています。USBケーブルを使用してコンピュータに接続するか、AC-DCアダプタまたはバッテリで電源を供給するだけです。

互換性

  • Arduino Duemilanoveプロトタイプシールド
  • Arduino I / Oシールド
  • Arduino LCDシールド
  • Arduinoモーターシールド
  • Arduinoイーサネットシールド

仕様

  • マイクロコントローラ：ATmega328
  • 入力電圧（推奨）：7-12V
  • 入力電圧（リミット）：6～20V
  • アナログ入力ピン：8
  • DC電流：40mA（I / Oピン）、50mA（3.3Vピン）
  • フラッシュメモリ32 KBのうち、ブートローダが使用する2 KB
  • SRAM：1KB（ATmega168）または2KB（ATmega328）
  • EEPROM：512バイト（ATmega168）または1KB（ATmega328）
  • クロック速度：16MHz
