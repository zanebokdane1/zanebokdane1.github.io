CORSAIR AirGuide テクノロジー
SP RGB Elite series ファンは、アンチボルテックス羽根を使って強力なエアフローと集中的な冷却を行っており、吸入、排気として使用する場合でも、水冷ラジエーターまたはヒートシンクへの取り付ける場合でも冷却を改善します。
8 個の個別アドレス可能 RGB LED が中央ファンハブに並び、半不透明なファンブレードを通して外側に拡散し、繊細でエレガントなグロー効果を実現します。
セットアップを思いのままに
付属*の iCUE Lighting Node CORE によって、鮮やかなライティング効果を作り出し、強力な iCUE を使用してダイナミックな RGB ライティングでシステムに息吹を吹き込むことができます。
* Lighting Node Core はマルチパックのみに含まれますが、別途購入できます
